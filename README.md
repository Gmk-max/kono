## 技术栈

以 **Spring Boot 2.3.x.RELEASE** 为基础，涉及的框架和中间件有：

   + springboot 基础整合框架
   + servlet4  web 标准
   + undertow 或者tomcat   web 容器
   + spring cache  缓存抽象层
   + spring security  安全框架
   + json web token  安全框架**token**技术
   + mybatis plus 3  **ORM**增强
   + spring data jpa (选)
   + redis  缓存中间件
   + mysql   数据库
   + mapstruct  bean转换器，编译期使用
   + lombok  bean简化工具
   + swagger2 文档（开发测试）
   + docker 容器技术
